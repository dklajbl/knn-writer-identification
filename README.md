# Identification of writers by handwriting.

**Code:**

- Function descriptions
```
def add(a, b):
    """
    Add two numbers.

    Parameters:
        a (int): First number.
        b (int): Second number.

    Returns:
        int: Sum of a and b.
    """
    return a + b
print(add(3, 7))
```

- Naming conventions:
    - premenne, funkcie: snake_case
    - triedy: CamelCase
